# UID_lab2

Десктопний застосунок, розроблений за допомогою фреймворку **Electron**. Програма являє собою інтерактивну форму опитування, яка збирає інформацію про останню прочитану користувачем книгу та зберігає результати у локальний файл.

---

##  Функціонал

* **Послідовні запитання**: Програма по черзі виводить 5 тематичних запитань про книги.
* **Валідація введення**: Реалізовано перевірку на порожнє поле — застосунок не дозволить надіслати пусту відповідь.
* **Збереження даних**: Кожна відповідь автоматично записується у файл Media/answers.txt разом із текстом відповідного запитання.
* **Динамічний інтерфейс**: Використання фонового зображення, стилізованих шрифтів та напівпрозорих елементів для комфортної роботи.
* **Завершення сесії**: Після надання відповіді на останнє запитання кнопка блокується, а користувач бачить повідомлення про завершення опитування.

---

##  Технологічний стек

* **Ядро**: Node.js.
* **Фреймворк**: Electron (v40.6.0).
* **Frontend**: HTML5, CSS3.
* **Логіка**: JavaScript (використання модуля fs для роботи з файловою системою).

---

##  Структура проєкту

* main.js — головний процес Electron; відповідає за створення та налаштування вікна додатка.
* index.html — візуальна структура інтерфейсу та CSS-стилі.
* renderer.js — логіка обробки подій, керування масивом запитань та запис відповідей у файл.
* package.json — маніфест проєкту, що містить список залежностей та скрипти запуску.
* Media/ — папка для ресурсів. Містить фонове зображення background.jpg та файл результатів answers.txt.
