# UID_lab2
Lab2
Десктопний застосунок «Форма опитування», розроблений з використанням Electron та Node.js. 
Програма відображає користувачу послідовність запитань, дозволяє вводити відповіді у текстове поле та зберігає їх у текстовий файл Media/answers.txt на диску. 
Інтерфейс реалізовано за допомогою HTML і CSS, логіка обробки подій — на JavaScript, з використанням модуля fs для роботи з файловою системою.
