# UID_lab2
Опитування про прочитані книги (Electron App)
UID_lab2 — це десктопний застосунок, розроблений за допомогою фреймворку Electron. Програма являє собою інтерактивну форму опитування, яка збирає інформацію про останню прочитану користувачем книгу та зберігає результати у локальний файл.

 Функціонал
Послідовні запитання: Програма виводить 5 тематичних запитань по черзі.

Валідація введення: Заборона надсилання порожніх відповідей за допомогою сповіщень.

Збереження даних: Кожна відповідь записується у файл Media/answers.txt разом із текстом запитання.

Динамічний інтерфейс: Використання фонового зображення та сучасних стилів для зручного користування.

Завершення сесії: Після останнього запитання кнопка стає неактивною, а користувач бачить фінальне повідомлення.

 Технологічний стек
Ядро: Node.js

Фреймворк: Electron (v40.6.0)

Frontend: HTML5, CSS3

Логіка: JavaScript (модуль fs для роботи з файловою системою)

 Структура проєкту
main.js — головний процес Electron, відповідає за створення вікна додатка.

index.html — структура інтерфейсу та стилізація форми.

renderer.js — логіка обробки подій, керування масивом запитань та запис у файл.

package.json — конфігурація проєкту та залежності.

Media/ — папка для медіа-файлів (тут має бути background.jpg та створюється answers.txt).
